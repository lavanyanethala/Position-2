# Image Hover Effect Project

This project is created using HTML and CSS.

## Main Concept
The main concept used in this project is CSS Position.

- The container is set to `position: relative`.
- The text (`p`) is set to `position: absolute`.
- This helps the text appear on top of the image.

## Hover Effect
- When the user hovers on the container:
  - The image becomes *blurred*.
  - The text *slides from right to left* and becomes visible.

## Technologies Used
- HTML
- CSS

This project is useful for learning **CSS Position, Hover Effects, and Transitions**.
